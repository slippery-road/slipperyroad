# SLIPPERY ROAD – sito finale funzionante

Questa è la versione corretta e pronta da caricare su GitHub Pages.

Contiene:
- logo SVG cliccabile che riporta alla Home;
- foto della band nella Home senza tagli aggressivi;
- foto dei componenti nella sezione Band;
- sezione Tour con locandine JPG;
- sezione Merch con magliette, cappelli e portachiavi;
- sezione Media con gallery cliccabile;
- layout responsive per PC e smartphone;
- fix della lightbox: non rimane più aperta all’avvio.

## File principali
- `index.html` → struttura del sito
- `style.css` → colori, dimensioni, responsive, layout
- `script.js` → dati modificabili: membri, date tour, merch e media
- `assets/` → logo, immagini, locandine, merch, gallery

## Caricamento su GitHub
Estrai lo ZIP e carica nella root del repository questi elementi:

```text
index.html
style.css
script.js
README.md
robots.txt
sitemap.xml
assets/
```

Quando apri il repository su GitHub devi vedere subito `index.html`, non una cartella esterna che contiene il sito.
